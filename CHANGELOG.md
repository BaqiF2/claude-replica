- 🎉 初始版本发布

- 🔧 Bash 命令执行

- 交互式模式支持

- 命令行参数解析

- 🎯 技能系统（Skills）- 自动加载领域知识

- 🔌 插件系统（Plugins）- 打包的功能扩展

- ⏪ 回退系统（文件快照和恢复）
- 🖼️ 图像支持（PNG、JPEG、GIF、WebP）
- 🏭 CI/CD 支持（自动检测 CI 环境）

#### 配置系统
- 多级配置支持（用户、项目、本地）

#### 其他功能
- 上下文管理和压缩

## 版本说明

### 版本号格式

`主版本号.次版本号.修订号`

- **主版本号**: 不兼容的 API 变更
- **次版本号**: 向后兼容的功能新增
- **修订号**: 向后兼容的问题修复

### 变更类型

- **新增**: 新功能
- **变更**: 现有功能的变更
- **弃用**: 即将移除的功能
- **移除**: 已移除的功能
- **修复**: Bug 修复
- **安全**: 安全相关的修复
